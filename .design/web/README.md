# Web (호스트) 문서 — 최신

이 레포의 `apps/web`은 **Playground를 호스팅하는 최소 Next.js 앱**입니다.  
웹사이트/회원/결제/백엔드는 이 레포 범위에 포함되지 않습니다.

## 현재 범위
- `/playground` UI 제공
- `@robota-sdk/playground` 패키지(라이브러리)를 소비하는 호스트 역할

## 현재 범위가 아닌 것(존재하지 않음)
- 인증/회원가입/로그인
- Firebase/Stripe/크레딧/요금제/구독
- 웹사이트(홈/대시보드/프로필/설정 등)
