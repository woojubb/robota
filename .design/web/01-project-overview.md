# Robota Web / Playground 개요 (최신)

이 레포의 웹 영역(`apps/web`)은 **Playground를 호스팅하는 최소 앱**이며, 제품 형태는 **SaaS 웹사이트가 아니라 Playground UI**입니다.

## 현재 범위
- **Playground UI**: 워크플로우를 시각화하고 실행 결과를 확인하는 UI
- **SDK 중심 구조**: 실행/이벤트/워크플로우 모델은 SDK 패키지가 소유하고, web은 이를 소비한다

## 제거된 범위(현재 존재하지 않음)
- 사용자 인증/회원가입/로그인
- Firebase 기반 백엔드
- 결제/크레딧/요금제/구독
- “팀 협업 플랫폼” 전제의 제품 기능

## 핵심 원칙
- 모든 agent는 동일한 생명주기/이벤트 규칙을 따른다(특수 agent 개념 없음).
- 관계/연결은 `context.ownerPath` 기반 규칙을 따른다(추론/대체 경로 없음).

## 기술 스택(현재)
- Next.js(호스트)
- TypeScript
- Tailwind CSS
- React Flow 기반 시각화(UI)
- **AI 에이전트 시장 급성장**: 2024년 대비 300% 성장 예상
- **워크플로우 시각화 도구 부족**: 기존 솔루션의 한계 극복
- **개발자 도구 시장**: 개발 생산성 향상에 대한 높은 수요

## 프로젝트 현재 상태

### ✅ 완료된 핵심 기능 (100%)
- 실시간 워크플로우 시각화 시스템 완전 구현
- AssignTask 분기 구조 100% 지원
- Mermaid 다이어그램 실시간 생성 및 렌더링
- 23개 WorkflowNode, 34개 Connection 완전 구조

### ⏳ 다음 단계 (Phase 5)
- React 컴포넌트 개발 및 웹 플랫폼 통합
- WebSocket 실시간 통신 구현
- 사용자 인터페이스 완성

## 성공 지표

### 기술적 지표
- **실시간 성능**: 워크플로우 업데이트 < 50ms
- **확장성**: 100+ 노드 워크플로우 지원
- **안정성**: 99.9% 시스템 가용성

### 비즈니스 지표
- **사용자 증가**: 월 10,000+ 활성 사용자
- **수익 목표**: 연간 $1M+ ARR
- **고객 만족도**: NPS 50+ 달성

Robota SaaS 플랫폼은 AI 에이전트 개발의 복잡성을 해결하고, 개발자들이 더 효율적으로 AI 시스템을 구축할 수 있도록 돕는 혁신적인 플랫폼입니다.
