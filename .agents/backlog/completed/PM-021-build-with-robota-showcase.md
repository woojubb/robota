## Background

"실제로 뭔가 만들어진다"는 증거가 SDK 통합 개발자 페르소나의 채택에 결정적이다. 현재 robota SDK로 만들어진 프로젝트/제품이 전혀 소개되지 않는다.

## 작업 항목

- `robota.io/showcase` 페이지 생성
- 초기 쇼케이스 3개 직접 발굴:
  - 내부 사용 사례 (이 프로젝트 자체 개발)
  - SDK 임베딩 예제 앱 (PM-008)
  - 커뮤니티 기여 프로젝트
- 투고 폼 제공 (프로젝트명, 설명, 링크, 사용한 패키지)
- 각 쇼케이스에: 스크린샷, 사용된 robota 패키지, 소스 링크
- 월 1회 "이달의 프로젝트" 선정

## Test Plan

- 투고 폼 제출 확인
