## Background

Product Hunt 런치는 초기 MAU 급증의 단일 최대 기회다. 준비 없이 실행하면 역효과. PM-001 온보딩 수술과 PM-002 비교 페이지가 완료된 후에 실행해야 한다.

## 작업 항목

- Product Hunt 계정 생성 + 제품 페이지 작성
- 런치 자료 준비:
  - 스크린샷 5장 (설치 → 첫 실행 → 실제 코딩 → 멀티 공급자 → Web Playground)
  - 시연 GIF 2개 (온보딩 60초, 실제 사용 90초)
  - 태그라인: "The open-source AI coding CLI. Any provider. BYOK."
- 런치 당일 계획:
  - 한국 시간 기준 오전 1시 런치 (Product Hunt 기준 자정)
  - 커뮤니티 알림: Discord, GitHub Discussions, LinkedIn, X
- 런치 전 베타 사용자 100명 이상 확보 조건 설정

## Test Plan

- 런치 전 자료 완성도 체크리스트 확인
- 런치 후 48시간 트래픽 모니터링
