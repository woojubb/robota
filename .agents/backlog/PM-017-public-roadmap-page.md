## Background

"이 프로젝트가 어디로 가는지"를 보여주는 공개 로드맵이 없다. 오픈소스 신뢰의 핵심 요소이며, 장기 사용자 형성의 기반이다.

## 작업 항목

- `robota.io/roadmap` 페이지 생성
- 3단계 뷰: 현재 개발 중 / 다음 분기 예정 / 검토 중
- GitHub Issues 투표 기능 연동 (👍 반응 수 집계)
- 분기별 업데이트 (1~3월, 4~6월, 7~9월, 10~12월)
- 구독 기능: 이메일로 로드맵 업데이트 알림
- 완료된 항목은 Changelog 연결
