## Background

Hacker News Show HN은 비용 없이 기술 개발자에게 도달하는 가장 효율적인 채널이다. "Claude Code 대안 오픈소스"라는 포지셔닝이 명확할 때 HN 공동체에 공명한다.

## 작업 항목

- Hacker News Show HN 게시 준비
  - 제목: "Show HN: robota – open-source AI coding CLI, any provider, BYOK"
  - 댓글 대응 준비: FAQ 10가지 사전 작성
- Reddit 포스팅: `r/LocalLLaMA`, `r/AItools`, `r/typescript`
- X(트위터) 개발 스레드: 제작 과정 + 차별점 스레드 형식
- 각 플랫폼에 맞는 포지셔닝 메시지 차별화
  - HN: 기술 아키텍처 + SDK 이중성 강조
  - Reddit r/LocalLLaMA: 로컬 LLM 지원 강조
  - Reddit r/typescript: TypeScript SDK 강조
