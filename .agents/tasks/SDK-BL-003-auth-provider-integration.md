## 요약

Anthropic이 서드파티용 공식 OAuth를 제공하면 적용. 현재는 API Key만 가능하며 OAuth는 Claude Code/Claude.ai 전용으로 차단됨.

## 트리거 조건

Anthropic이 서드파티 OAuth Provider 역할을 공식 발표하면 이 태스크를 active로 전환.
