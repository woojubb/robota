# GUI-002 — agent-gui Tauri + sidecar foundation (Stage 1 MVP)

Spec: `.agents/spec-docs/todo/GUI-002-agent-gui-tauri-sidecar-foundation.md` (status: approved → active on
GATE-IMPLEMENT). Owner-approved 2026-07-12 ("Approve — build Stage 1 MVP"). Design ENDORSED (3 review rounds).

Build a thin **Tauri v2** desktop shell that spawns a **Node `robota` sidecar** over a loopback WS port with
a **required launch-nonce auth**, and loads the existing **agent-web-ui** React SPA in its webview — no
session/command/permission logic in the GUI.
