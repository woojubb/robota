# Daily work reports (OBSERVABILITY-001)

One **committed** markdown report per **UTC day that had work** (git commits), named `YYYY-MM-DD.md`. This is
a durable, reviewable record of what was done each day. No-work days have no report. (Note: this lives at
`.agents/daily-reports/`, NOT the gitignored `.agents/reports/` throwaway analysis dir.)

Generated by the daily-report harness and enriched by an agent:

- **Harness** (`scripts/harness/daily-report.mjs`) — fills the factual sections (commits, merged PRs,
  files/specs/tasks touched) from git history, catching up day-by-day from the last report, skipping
  no-work days.
- **Agent** (the [`daily-report`](../skills/daily-report/SKILL.md) skill) — authors the `## Summary`
  prose and commits the report.

Run manually: `node scripts/harness/daily-report.mjs --plan` (preview) or `node scripts/harness/daily-report.mjs`
(write). See the skill for the background-trigger flow (fires at a UTC hour boundary during active work).
