---
name: Design confirmation required
description: All architecture/design proposals must get explicit user approval before implementation
type: feedback
---

설계안은 반드시 사용자 컨펌을 받아야 한다. 설계 문서를 작성한 후 바로 구현으로 넘어가지 말 것.

**Why:** 설계 결정은 프로젝트 방향에 큰 영향을 미치므로 사용자가 검토하고 승인해야 함.

**How to apply:** 설계 문서 작성 → 사용자에게 리뷰 요청 → 승인 후에만 구현 시작. "구현 시작할까요?"가 아니라 "설계안을 검토해주세요"로 마무리.
