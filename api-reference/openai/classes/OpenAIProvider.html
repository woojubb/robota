▸ chatStream(model, context, options?): AsyncGenerator<StreamingResponseChunk, void, unknown>

Similar to chat() but returns an async iterator that yields response chunks as they arrive from OpenAI's streaming API. Useful for real-time display of responses or handling large responses incrementally.
