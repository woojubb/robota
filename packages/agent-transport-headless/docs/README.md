# agent-transport-headless Docs Index

`@robota-sdk/agent-transport-headless` exposes `InteractiveSession` through non-interactive print/headless execution.

## Current Capabilities

- Text, JSON, and stream-json output modes for `robota -p`.
- Deterministic headless behavior with no startup update checks.
- Background task controls through the shared SDK/runtime session layer.
