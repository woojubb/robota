### Examples

Package-owned examples for `@robota-sdk/agents`.

#### Files
- `basic-conversation.ts`: Basic Robota conversation flow.
- `tool-calling.ts`: Tool calling usage.
- `multi-providers.ts`: Multi-provider comparison; expand beyond OpenAI.
- `advanced-features.ts`: Advanced features such as LoggingPlugin.
- `agent-templates.ts`: Multiple Robota configurations; split by provider if needed.
- `execution-analytics.ts`: ExecutionAnalyticsPlugin example; avoid `as any`.
- `agents-basic-usage.ts`: Agents basic usage and plugins.
- `agents-streaming.ts`: Agents streaming flow.
