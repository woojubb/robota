# @robota-sdk/playground

Deployable Playground UI package for Robota SDK.

## Overview

- Implementation is React-based.
- This repository keeps a minimal Next.js host (`apps/web`) that renders only `/playground`.
- Authentication, signup/login, credits, billing, and pricing are intentionally not included.

## Notes
- Monaco IntelliSense/type injection is disabled; the playground is editor-only.

## Recent Updates
- Removed Monaco IntelliSense injection and related stubs.
- Aligned workflow bridge usage with the shared SSOT implementation.
