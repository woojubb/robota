# Execution Tree Testing Guide

## Testing the Real-Time Execution Tree System

### Method 1: Dedicated Test Page

Navigate to `/execution-tree-test`.

This page provides:
- demo generation actions
- tree visualization
- block detail inspection
- debugging support

### Method 2: Main Playground

1. Open `/playground`.
2. Find the block visualization panel.
3. Open the debug-oriented view.
4. Trigger a demo scenario and inspect the tree output.

## What to Inspect

- parent-child relationships
- execution levels
- execution paths
- timing data
- block metadata
- status counts
