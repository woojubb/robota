# DAG Studio App

Dedicated Next.js application for the DAG Designer interface.

- Hosts the `/dag-designer` route
- Runs on port 3001 (dev mode) to avoid conflict with `apps/web`
- Depends on `@robota-sdk/dag-designer` and `@robota-sdk/dag-core`
